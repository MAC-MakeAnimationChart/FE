# MAC (Make A Chart)

비개발자도 데이터만 넣으면 차트를 만들고 수정할 수 있는 한글형 No-Code 시각화 웹서비스

## 주요 기능

### ✅ 구현 완료

1. **CSV 파일 업로드**
   - 드래그 앤 드롭 또는 파일 선택으로 데이터 업로드
   - 자동으로 컬럼 감지 및 데이터 파싱

2. **데이터 테이블 뷰**
   - 엑셀과 유사한 표 형태로 데이터 표시
   - 셀 클릭으로 데이터 직접 수정 가능
   - 실시간 데이터 업데이트

3. **막대 그래프 시각화**
   - Apache ECharts 기반 고품질 차트
   - 반응형 차트 렌더링
   - 실시간 미리보기

4. **차트 커스터마이징**
   - **데이터 설정**
     - Top N 행 수 제한
     - 카테고리/값 컬럼 선택
   
   - **색상 설정**
     - 막대 색상
     - 배경 색상
   
   - **막대 스타일**
     - 두께 조절
     - 투명도 조절
     - 간격 조절
   
   - **글자 서식**
     - 라벨 크기 및 굵기
     - 축 글자 크기 및 굵기
   
   - **범례 (Legend)**
     - 표시/숨김
     - 글자 크기 및 굵기
   
   - **제목**
     - 대제목 (크기, 굵기, 위치)
     - 부제목 (크기, 굵기, 위치)
   
   - **출처**
     - 데이터 출처 표시
     - 글자 크기 조절

5. **내보내기**
   - JPG 형식으로 차트 다운로드
   - 고해상도 (1920x1080) 이미지 생성

## 사용 방법

1. **데이터 업로드**
   - "데이터" 탭에서 CSV 파일 업로드
   - 자동으로 "미리보기" 탭으로 전환

2. **차트 설정**
   - 우측 설정 패널에서 카테고리와 값 컬럼 선택
   - 원하는 스타일로 차트 커스터마이징

3. **내보내기**
   - 설정 패널 상단의 "내보내기" 버튼 클릭
   - JPG 파일로 다운로드

## 기술 스택

- **Frontend**: React + TypeScript
- **Chart Library**: Apache ECharts
- **CSV Parsing**: PapaParse
- **UI Components**: Radix UI + Tailwind CSS
- **Notifications**: Sonner

## 향후 개발 예정

- 다양한 차트 타입 (Line, Pie, Donut, Stacked Bar, Bar Chart Race)
- 차트 추천 기능
- 온라인 저장 및 공유
- 애니메이션 효과
